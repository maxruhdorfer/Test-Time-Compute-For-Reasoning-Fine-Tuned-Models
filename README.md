# Test-Time-Compute-For-Reasoning-Fine-Tuned-Models
In this repository we study which test-time strategy maximizes accuracy for a fixed inference compute budget — and if the answer depends on whether the model was fine-tuned with SFT vs. GRPO. We study a range of state-of-the-art compute time strategies: i) Majority Voting ii) Best of N with PRM iii) PRM guided Beam Search iv) Budget forcing.
